# book-list-client

**Author**: Sunitha & Gregor
**Version**: 0.3.0

## Overview
This mobile app displays a list of books from our database and allows users to add new books.

## Getting Started
npm install
server: nodemon index.js
database: pgstart
front-end: live-server

## Architecture
Javascript
JQuery
Node.js
